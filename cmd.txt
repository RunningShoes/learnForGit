	git初始化：
		git init

	git查看历史状态：
		git status：查看仓库的状态，可以看看那些文件被修改了
		git diff <file>:查看某个文件具体的修改

	git 提交代码：
		git add <file>
		git commit -m "注释"

	查看提交日志：
		git log

	版本回退：
		git reset --hard HEAD^<num>//代表回退数目
		git reset --hard commit_id   //回到当时的版本
		git relog   //回退时找不到log的方法
